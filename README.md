# Todo App ✅

このアプリは、React（TypeScript）を使って作成したToDoリストです。  
未経験からエンジニアを目指している中で、基礎技術や状態管理の理解を深めるために作成しました。

## 📝 主な機能
- タスクの追加・削除・編集
- チェックボックスでの 完了・未完了 の管理

## 💡 学習目的・工夫した点
- 型定義やPropsの使い方を幅広く理解するために TypeScriptを使用したTodoアプリを作成
- コンポーネント分割と状態管理の実践
- GitHub Pagesを用いたデプロイを経験
